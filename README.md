📊 ML Algorithm Visualizer (C++ Project)
📌 Overview

This project is a simple Machine Learning Algorithm Visualizer built using C++, focusing on OOPs and Data Structures.
It simulates basic ML workflows like data handling, training process, and prediction in a structured way.

⚙️ Features
Object-Oriented design using classes
Use of STL Data Structures (vector, map, etc.)
Basic ML pipeline simulation
Training process visualization (console-based)
Easy to extend for more algorithms
🛠️ Tech Used
C++
OOP (Classes, Inheritance, Polymorphism)
Data Structures (STL)
Basic Algorithm Simulation
